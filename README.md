# API de Jogos Favoritos

API REST criada para gerenciamento de jogos favoritos dos usuários.

## Endpoints

- GET /games → Lista todos os jogos
- GET /games/{id} → Busca um jogo por ID
- POST /games → Cadastra um novo jogo
